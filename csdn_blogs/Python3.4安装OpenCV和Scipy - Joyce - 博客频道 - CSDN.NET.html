Python3.4安装OpenCV和Scipy

标签： pythonopencv
2017-05-04 17:05 22人阅读 评论(0) 收藏 举报
本文章已收录于：
分类：

Python是个很方便的东西，脚本语言，只需要一个解释器就可以运行，比Matlab还方便，用来简单处理一些文本，图像之类的东西非常方便。
如果是要处理图像，OpenCV应该是很多人的第一选择。
安装步骤如下：

  • 安装pip，pip是个很方便的工具，类似于Linux下的apt。首先在命令行中打开Python安装路径下的Scripts文件夹，在该文件夹下执行如下命令：
    $ easy_install.exe pip
    然后就自动装好并且设置好环境变量了。
  • 安装基础运算库numpy：
    $ pip install numpy
  • 安装图形处理库Matplotlib：
    $ pip install matplotlib
  • 安装gcc，因为OpenCV是用C和少量C++写的，Windows用MinGW
  • 安装科学运算库Scipy，windows下这个不能简单的用pip install完成，因为这个库并不能在Windows下编译，需要自己下载编译好的轮子安装，这里给出网址，请自行选择适合自己机器的版本：
    http://www.lfd.uci.edu/~gohlke/pythonlibs/#scipy
  • 安装OpenCV，Windows下这个需要自行去OpenCV官网下载，书上和网上给的网址一般是http://opencv.org/downloads.html，其实这个是不对的，可能是好久之前的页面了，现在是http://opencv.org/releases.html，也可以直接百度OpenCV，进入官网首页，点击RELEASES超链接过去。
  • 下载完之后，双击安装，在安装路径下的 build/python/2.7 路径有两个文件夹，分别对应64位和32位系统，里面有个cv2.pyd，复制到Python安装路径的 \Python34\Lib\site-packages 路径下，就可以了。

虽然文件夹名称是2.7，本来应该对应Python2.7，但是测试发现Python3.4也是可以用的。

然后我们来测试一下。测试代码如下：

import cv2

img = cv2.imread('F:/Picture/2012030921261518338.png')
cv2.imshow('Itisatitle',img)
cv2.waitKey()
cv2.destroyAllWindows()
